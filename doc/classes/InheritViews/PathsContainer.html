inherit_view_paths= (new_paths)

Set the inherit view paths, in order of self to ancestor. The controller_path for self is always prepended to the front, no matter what the arguments.

[show source] 
# File lib/inherit_views.rb, line 86
    def inherit_view_paths=(new_paths)
      new_paths -= [controller_path]
      old_paths = inherit_view_paths - [controller_path] - new_paths
      instance_variable_set('@inherit_view_paths', [controller_path] + new_paths + old_paths)
    end
